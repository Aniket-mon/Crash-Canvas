# Crash Canvas

A Road Accident Analysis and Prediction System for data-driven traffic safety insights.

## Overview

Crash Canvas analyzes historical accident data to identify patterns, predict high-risk areas, and provide actionable insights for improving road safety through machine learning and spatial analysis.

## Features

- **Data Processing**: Clean and process accident datasets
- **Spatial Analysis**: QGIS integration for hotspot mapping
- **Predictive Modeling**: Machine learning for risk prediction
- **Visualization**: Interactive dashboards and maps
